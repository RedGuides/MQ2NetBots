# /netnote

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/netnote <custom note>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
A custom setting to describe what your characters are doing.
<!--cmd-desc-end-->

## Examples

- <code>/netnote kissing a kobold</code>
- <code>/echo ${NetBots[Toonname].Note}</code> returns, "kissing a kobold"
